- 修正了 moontv 项目网页无法显示的问题，修复点包括路由 Handler 名称不匹配、缺少子目录支持及静态文件相对路径问题。
- 已交付修复后的源码及一键编译脚本 (build.sh)，验证编译通过。
- 更新了 GitHub Actions 工作流，添加标签触发机制和部署包打包步骤，确保输出包含二进制和静态文件目录。
- 修复了 GitHub Actions 编译时的 Rust 生命周期错误 (E0700)，将 `serve_file` 函数参数从 `&str` 改为 `String` 类型。
- 修复了 GitHub Actions 编译时的 Rust 生命周期错误 (E0515)，将 `get_mime_type` 函数返回类型改为 `&'static str`。

- 修复了管理员权限判断失效的问题，原因是后端返回角色为字符串 "admin" 而前端使用数字 1 进行比较，已统一改为字符串比较。
